Getting Started

Your journey from Figma pixels to production-ready code starts here. Let's build something sketchy but solid.

terminal

Quick Install

Grab the CLI and start translating designs instantly.

npm install -g figco-cli

The Flow

Select a frame, run the command, get the raw HTML/Tailwind output. No magic, just math.

Auth with Figma

You need to tell FigCo where to look. Run the auth command and paste your personal access token.

figco auth --token YOUR_TOKEN_HERE
2

Export a Frame

Target a specific node ID from your Figma file URL.

figco export https://figma.com/file/... --node 12:34
